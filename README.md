# Office Tomato Hunt

React/Vite mini-game.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Azure Static Web Apps

Use:
- App location: `/`
- Output location: `dist`

## PayPal Donate

Replace `YOUR_BUTTON_ID` in `src/App.jsx` with your PayPal donate button id.
